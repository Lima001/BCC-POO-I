inlineprotected

Método usado para alocar memória dinâmica da Matriz que o Invocar.

Caso seja necessário alterar a estrutura da Matriz, será necessário realocar a quantidade de memória disponível para ela.

Uma vez alocados os espaços na memória para a Matriz, os valores que podem ser interpretados como linhas e colunas serão todos preenchidos por 0.

Parâmetros
n_linha_usado para definir a nova quantidade de linhas da Matriz e permir a alocação de Memória
n_coluna_usado para definir a nova quantidade de colunas da Matriz e permir a alocação de Memória

◆ copiar()

void Matriz::copiar ( const Matriz &  m)
inlineprotected

Método usado para copiar os elementos de uma Matriz.

Esse método é usado para copiar apenas os elementos de uma Matriz, isso é os valores de presentes em suas linhas e colunas. Para isso observe que a Matriz que terá a sua estrutura copiada deve obrigatoriamente conter a mesma quantidade de linha e coluna da Matriz de destino.

Parâmetros
mMatriz que terá seus elementos copiados

inline

Método utilizado para preencher uma linha inteira da Matriz com valores.

Esse método é usado para tornar a atribuição de valores a Matriz mais cômoda ao desenvolvedor. Através desse método é possível definir os valores de uma linha inteira através de um vetor C++ (vector). Os valores serão copiados conforme sua posição no vetor, "da esquerda para a direita".

Parâmetros
indexíndice da linha que será alterada - Deve ser um valor >= 0 e dentro dos limites da estrutura da Matriz
linhavector contendo os elementos das colunas daquela linha
Aviso
Caso o índice informado não seja válido na Matriz, bem como caso a quantidade de elementos do vetor seja superior a suportada pelo objeto, a operação não será efetuada e o método será finalizado

Método usado para resetar a Matriz a um estado básico sem linhas nem colunas.

Esse método é usado principalmente na necessidade de realocar a quantidade de memória dinâmica disponível para a Matriz. Para isso é necessário antes deletar toda a memória anteriormente alocada para a Matriz para possível configuração conforme contexto.

Múltiplicação entre Matrizes.

Realiza a múltiplicação entre duas matrizes gerando ao final um novo objeto do tipo Matriz como resultado da operação.

Observação: Deve-se observar a condição matemática necessária para que a multiplicação entre as matrzies ocorra com sucesso. Considere uma matriz \( A (linha X coluna) \) e a matriz \( B (linha X coluna) \). Para que a multiplicação ocorra deve-se observar a seguinte verdade \( A coluna = B linha \)

Parâmetros
mReferência a Matriz usada na operação de múltiplicação
Retorna
Matriz resultante da múltplicação de matrizes

Atribuição Move.

Movimenta a estrutura completa de uma Matriz. Dessa forma pode-se usar como parâmetro qualquer Matriz, independente de sua quantidade de linhas e colunas divergirem da quantidade atual da Matriz que invocou o método. Isso deve-se ao fato da alocação dinâmica de memória realizada na chamada desse método.

Parâmetros
vetorReferência rvalue da Matriz Base para a operação
Retorna
Referência para o Matriz que Invocou o Método

Operador de Acesso.

Usado quando é necessário acessar um elemento em um índice específico da Matriz. Permite tanto a leitura quanto a escrita da posição da Matriz .

Observação: O índice informado deve ser compatível com a estrutura da Matriz, caso contrário a operação não é realizada a fim de evitar acesso a espaços de memórias não pertencetes a Matriz .

Parâmetros

Retorna
Referência para o endereço de memória do elemento no índice informado
