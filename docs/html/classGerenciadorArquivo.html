◆ recuperarObjetos()

void GerenciadorArquivo::recuperarObjetos ( Objeto *  lista_objetos)
inline

Recupera todos Objeto do arquivo.

Esse método é usado para recuperar os Objeto presentes no arquivo passado para o gerenciador. Esse método é útil, pois permite a inicialização de diversos objetos usando como base um arquivo.

Observação: Devido ao fato de um Objeto possuir um atributo a que referese-se a um objeto Aceleracao e que esse obejto é constítuido por funções e seus endereços de memória no momento da instaciação, não é possível recuperar a função usada nos objetos. Dessa forma, todo Objeto recuperado de um arquivo terá sua aceleração definida pelo construtor default de Aceleracao, devendo o desenvolvedor alterar esse atributo posteriormente conforme necessidade.

Parâmetros
lista_objetosPonteiro para um array de Objeto que deve possuir alocado ao mínimo a quantidade de Objeto salvos no arquivo para que esses possam ser recuperados e inseridos nesse array
