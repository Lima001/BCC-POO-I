Projeto desenvolvido para apresentação na disciplina de Programação Orientada a Objetos I - Bacharelado em Ciência da Computação - IFC Blumenau
Lista de Classes
Aqui estão as classes, estruturas, uniões e interfaces e suas respectivas descrições:
 CAceleracao
 CAcelerecaoClasse usada para Definir as leis de aceleração que regem a movimentação de um Objeto tanto no sentido vertical quanto horizontal
 CCircunferenciaClasse para Representação de uma Circunferência. É a forma considerada como padrão para os objetos Objeto da simulação
 CControlador_SDLClasse responsável pela inicialização e possibilitar a operação da biblioteca SDL
 CCorRGBAClasse responsável por representar cores RGBA para uso em conjunto aos elementos gráficos da interface SDL
 CDetectorColisaoClasse para Cálculo e Aplicação de Colisão entre objetos do tipo Objeto e entre um Objeto e uma Linha
 CEventoClasse Responsável pela representação de um evento único capaz de ser processado por um objeto do tipo GerenciadorEvento
 CGerenciadorArquivoClasse para manipulação de arquivos visando a persitência de dados relacionados a objetos do tipo Objeto
 CGerenciadorEventoClasse Responsável pelo processamento dos objetos do tipo Evento
 CJanelaClasse responsável pela criação e controle da Janela gráfica e seus elementos associados
 CLinhaClasse para Representação de um Segmento de reta no Plano Cartesiano Usada para definição de limites nos quais os objetos da simulação colidem, e principalmente para o cálculo da reflexão especular
 CMatrizClasse para Representação de Matrizes
 CObjetoClasse para Representação de um Obejto com forma de Circunferência capaz de movimentar-se
 CPlanoCartesianoClasse responsável pela representação gráfica do Plano Cartesiano
 CPontoClasse para Representação de um Ponto no Plano Cartesiano
 CRelogioClasse responsável pelo controle dos FPS da aplicação gráfica
 CRenderizadorClasse responsável pela abstração do processo de renderização de figuras na Área de renderização da janela
 CRetanguloClasse para Representação de um Retângulo
 CTrianguloClasse para Representação de um Triangulo
 CVetorClasse para Representação de um Vetor no Plano Cartesiano
