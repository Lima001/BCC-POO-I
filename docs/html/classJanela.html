Construtor Default.

Usado para criar um objeto janela conforme parâmetros informados. Não incializa o ponteiro para SDL_Surface do objeto, sendo que esse deve ser inicializados somente através de seu métodos membros implementados, nas situações julgadas necessárias.

Em caso de erros durante o processo de criação da janela gráfica pelo SDL, o erro será exibido conforme retorno da função SDL_GetError().

Parâmetros
tituloUma string estilo C. É usado para criar a janela atavés da função SDL_CreateWindow() que solicita um título
larguraInicializa atributo de mesma nomenclatura
alturaInicializa atributo de mesma nomenclatura

Destrutor da Classe.

Verifica se alguma janela gráfica foi construída a partir do SDL. Em caso de existência de algo do tipo, a janela é destruída por esse método.

Funções membros

◆ atualizar()

void Janela::atualizar ( )
inline

Atualiza a Área de Renderização.

Esse método deve ser usado logo após todas as definições de renderização tiverem sido concluidas. Caso o método não seja chamado, não será possível observar os efeitos gráficos produzidos.

◆ iniciarSurface()

void Janela::iniciarSurface ( )
inline

Inicializa a área de renderização gráfica da janela.

Esse método é usado para inicializar o ponteiro ptr_surface que permite acesso a área de renderização gráfica da janela criada.

Observação: Deve-se somente chamar esse método se a janela gráfica SDL já tiver sido criada, podendo resultar erro em caso contrário.

◆ preencherFundo()

void Janela::preencherFundo ( const CorRGBA &  cor)
inline

Define uma cor de fundo para a Área de renderização gráfica da janela.

Método usado quando deseja-se resetar o gráfico da área de renderização usando-se da definição de uma cor de fundo padrão.

Parâmetros
corReferência constante para um objeto CorRGBA que será usado para gerar a cor compreendida pelas funções SDL e que representa a cor de fundo desejada pelo desenvolvedor
