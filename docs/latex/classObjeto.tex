Movimenta o \mbox{\hyperlink{classObjeto}{Objeto}} alterando possivelmente sua velocidade e posição. 

Esse método deve ser usado quando deseja-\/se calcular o movimento realizado pelo \mbox{\hyperlink{classObjeto}{Objeto}} em um determinado intervalo de tempo.

Atribuição por Cópia. 

Copia todos os atributos de um \mbox{\hyperlink{classObjeto}{Objeto}} passado como parâmetro para o \mbox{\hyperlink{classObjeto}{Objeto}} atual. Observe que a partir desse momento, o \mbox{\hyperlink{classObjeto}{Objeto}} atual torna-\/se \char`\"{}praticamente\char`\"{} o mesmo \mbox{\hyperlink{classObjeto}{Objeto}} copiado, tendo o mesmo id. Por essa razão não recomenda-\/se realizar a Atribuição por cópia e manter os dois Objetos na Simulação, uma vez que implicações no que tange a aplicação de colisão a partir do id pode gerar resultados inesperados.

Caso seja necessária uma \char`\"{}cópia\char`\"{} mais superficial onde somente os atributos gerais como raio, massa e atributos numéricos sejam copiados para o \mbox{\hyperlink{classObjeto}{Objeto}}, recomenda-\/se o uso da Atribuição por Movimentação, implementada já nessa classe.
