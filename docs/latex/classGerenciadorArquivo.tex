Recupera todos \mbox{\hyperlink{classObjeto}{Objeto}} do arquivo. 

Esse método é usado para recuperar os \mbox{\hyperlink{classObjeto}{Objeto}} presentes no arquivo passado para o gerenciador. Esse método é útil, pois permite a inicialização de diversos objetos usando como base um arquivo.

Observação\+: Devido ao fato de um \mbox{\hyperlink{classObjeto}{Objeto}} possuir um atributo {\itshape a} que referese-\/se a um objeto \mbox{\hyperlink{classAceleracao}{Aceleracao}} e que esse obejto é constítuido por funções e seus endereços de memória no momento da instaciação, não é possível recuperar a função usada nos objetos. Dessa forma, todo \mbox{\hyperlink{classObjeto}{Objeto}} recuperado de um arquivo terá sua aceleração definida pelo construtor default de \mbox{\hyperlink{classAceleracao}{Aceleracao}}, devendo o desenvolvedor alterar esse atributo posteriormente conforme necessidade.


\begin{DoxyParams}{Parâmetros}
{\em lista\+\_\+objetos} & Ponteiro para um array de \mbox{\hyperlink{classObjeto}{Objeto}} que deve possuir alocado ao mínimo a quantidade de \mbox{\hyperlink{classObjeto}{Objeto}} salvos no arquivo para que esses possam ser recuperados e inseridos nesse array \\
\hline
\end{DoxyParams}
